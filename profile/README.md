### 💡 Mission

At **Boksepyunsal**, we believe that technology should be a bridge, not a barrier. Our mission is to harness the power of Open Source to combat social inequality and information asymmetry. By democratizing access to high-quality tools and transparent information, we empower individuals to connect, grow, and compete on a truly equal footing.

### 🚀 Our Projects

#### 🤝 [Sumjima (숨지마)](https://github.com/Boksepyunsal/sumjima) — Zero-Fee Service Matching Platform
> *"Don't hide behind unfair fees. Your skills deserve better."*

An open-source service matching platform with **zero listing fees and zero commission**. Whether it's IT freelancing, home cleaning, tutoring, or professional consulting — every talent gets a fair shot.
**🔗 Live at [sumjima.fun](https://sumjima.fun/)**

`Next.js` `Supabase` `Tailwind CSS` `shadcn/ui` `Vercel`

---

#### 📋 [Text Util Extension](https://github.com/Boksepyunsal/text-util-extension) — Chrome Text Snippet Manager
A Chrome extension to save, organize, and sync your frequently used text snippets across all devices via your Google account. One-click copy, real-time search, and a clean modern UI — say goodbye to repetitive typing.

`React` `TypeScript` `Vite` `Tailwind CSS` `Chrome Extension MV3`

---

#### 🖥️ [BSPS MyBatis Formatter](https://github.com/Boksepyunsal/bsps-mybatis-formatter) — VS Code MyBatis XML Formatter
A VS Code extension that automatically formats SQL inside MyBatis XML mapper files. Configurable comma position, keyword casing, and indentation make it easy to enforce consistent SQL style across your team.

`VS Code Extension` `MyBatis` `SQL` `XML`

---

#### 📱 [re-qr](https://github.com/Boksepyunsal/re-qr) — QR Code Dashboard
A modern, responsive dashboard for generating and managing your personal QR codes. Secured with Kakao OAuth and Supabase Row Level Security — instantly turn any URL into a QR code.

`Next.js` `TypeScript` `Supabase` `Tailwind CSS` `shadcn/ui`

---

#### ⚡ [helpr](https://github.com/Boksepyunsal/helpr) — Rust CLI Script Manager
A cross-platform CLI tool to manage and execute your helper scripts from a single config file. Features fuzzy search, interactive parameter prompts, and works seamlessly on macOS, Linux, and Windows.

`Rust` `CLI` `Cross-platform`

### 🎯 Vision & Values

- **Fairness through Transparency**: We strive to create a world where information flows freely to everyone, ensuring no one is left behind due to a lack of resources or access.
- **Impactful Innovation**: We don't just build software; we build tools that make a tangible difference in people's daily lives and professional workflows.
- **Commitment to ESG**: We are deeply committed to **ESG (Environmental, Social, and Governance)** values. We believe that modern technology must be socially responsible, ethically governed, and sustainably developed to create a lasting positive impact on society.
